# INFO

This repository contains code used for the subject "Analisis y Evaluacion Sanitaria" of the 4th year of the Grade in Biology from the University of Oviedo

## Index

* AEB_Michaelis.R: contains code to generate Michaelis-Menten plots with different parameter
* AEBapp: contains code for a shiny app that allow students to interactively learn about diagnosis capacity of clinical tests, i.e. concepts of sensibility, specificity, ROC curves, reference intervals, etc...
